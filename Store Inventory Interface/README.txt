Store Inventory Interface

Overview
The Store Inventory Interface is a Java-based application that is a simulation developed using JavaFX in IntelliJ, designed to streamline store operations through an intuitive graphical user interface (GUI). Developed between January and May 2023, this application provides efficient management tools for pricing, sales tracking, inventory modifications, and customer management.

Features
- Graphical User Interface: Simplifies user interaction with an intuitive layout.
- Pricing Management: Easily manage and adjust pricing for store items.
- Sales Tracking: Monitor sales activities and generate sales reports.
- Inventory Management: Keep track of inventory changes and stock levels.

Prerequisites
Before running the application, ensure you have the following installed:
- Java Runtime Environment (JRE)
- JavaFX
- IntelliJ IDEA (recommended for development and running the application)



Open the project in IntelliJ IDEA or your preferred Java IDE.

Running the Application
To interact with the GUI, run the `ElectronicStoreApp` class. This can typically be done by navigating to the class file in your IDE and clicking the 'Run' button.

Usage
Upon launching `ElectronicStoreApp`, you will be greeted with the main interface of the Store Inventory System. From here, you can:
- Add or modify products in your inventory.
